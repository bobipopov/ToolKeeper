
# ToolKeeper

ToolKeeper е web приложение за управление на склад с инструменти.

## Features

- Добавяне на инструменти
- История на търсения
- Управление на склад
- Web интерфейс

## Tech Stack

- Next.js
- Supabase
- Vercel

## Demo

https://tool-keeper.vercel.app

## Installation

git clone https://github.com/bobipopov/ToolKeeper.git

cd ToolKeeper
npm install
npm run dev

## Environment

копирайте:

.env.example -> .env
